## Key Highlights
- **Canva AI 2.0 goes fully editable** — CPO Cameron Adams details a shift from one-shot generation to a collaborative editing environment trained on actual design-edit sequences, not just final outputs.
- **Agent frameworks dominate GitHub Trending** — OpenAI's `openai-agents-python` crosses 22k stars as multi-agent orchestration continues to consolidate around lightweight Python stacks.
- **Ambient-AI hardware surfacing** — BasedHardware's `omi` (AI that watches your screen and listens) is trending hard, a signal that "always-on assistant" is this cycle's form factor bet.
- Quiet Sunday across the major labs: no new posts from Anthropic, OpenAI, Google, DeepMind, NVIDIA, or TechCrunch since yesterday's digest.

## Analysis & Opinion
### [Inside Canva AI 2.0 with CPO Cameron Adams](https://www.therundown.ai/p/exclusive-inside-canva-ai-2-0-with-cpo-cameron-adams) — Rundown
Adams argues the interesting bit isn't the generation — it's that every generated element stays editable and the model keeps refining with you, trained on the *sequence* of edits real designers make rather than only finished artifacts. The framing positions AI as the execution layer while humans keep strategy, empathy, and audience judgment — a more honest division of labor than the "prompt-and-pray" generation tools that preceded it.

## New Products & Tools
### [openai/openai-agents-python](https://github.com/openai/openai-agents-python) — GitHub Trending
Lightweight Python framework for multi-agent workflows from OpenAI — 22.7k stars, ~470 added today. Becoming the reference implementation for agent orchestration in the OpenAI ecosystem.

### [BasedHardware/omi](https://github.com/BasedHardware/omi) — GitHub Trending
Ambient AI device + app that sees your screen and listens to conversations, then suggests actions. 10.7k stars, +609 today — ambient-assistant category is heating up.

### [thunderbird/thunderbolt](https://github.com/thunderbird/thunderbolt) — GitHub Trending
TypeScript framework pitched as "AI you control" — pick your own models, keep your own data, avoid vendor lock-in. +447 stars today; reflects ongoing pull toward self-hosted / BYO-model setups.

## Research
### [EvoMap/evolver](https://github.com/EvoMap/evolver) — GitHub Trending
Self-evolution engine for AI agents built around a "Genome Evolution Protocol." Biggest trending gainer of the day (+1,131 stars) — experimental territory, but worth watching as evolutionary approaches to agent improvement get more attention.
